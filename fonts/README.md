# خط ثمانية (Thmanyah) — مثبّت ✅

الملفات المستخدمة فعليًا في الموقع (صيغة woff2):

**Thmanyah Sans** — للواجهة والنصوص:
- `thmanyahsans-Regular.woff2` (400)
- `thmanyahsans-Medium.woff2` (500)
- `thmanyahsans-Bold.woff2` (700)

**Thmanyah Display** — للعنوان الكبير وأرقام العدّاد:
- `thmanyahserifdisplay-Regular.woff2` (400)
- `thmanyahserifdisplay-Bold.woff2` (700)
- `thmanyahserifdisplay-Black.woff2` (900)

الربط في أعلى `styles.css` داخل قواعد `@font-face`. لتغيير التوزيع، عدّل المتغيّرين `--font-body` و `--font-display` في `:root`.

> العائلة الكاملة (ستة أوزان × ثلاث عائلات: sans / serif text / serif display) والترخيص وأدلة الجماليات في مجلد التنزيل الأصلي.
